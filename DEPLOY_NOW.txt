================================================================================
                    METHOD 1: VERCEL (EASIEST - 5 MINUTES)
================================================================================

STEP 1: Push to GitHub
----------------------
Open your terminal and run these commands ONE BY ONE:

cd shakespeare-ga

git init

git add .

git commit -m "Initial commit - Shakespeare GA app"

Now create a GitHub repository:
1. Open: https://github.com/new
2. Repository name: shakespeare-genetic-algorithm
3. Click "Create repository" (don't add README)

git remote add origin https://github.com/YOUR-USERNAME/shakespeare-genetic-algorithm.git

git branch -M main

git push -u origin main

(Replace YOUR-USERNAME with your actual GitHub username)


STEP 2: Deploy on Vercel
-------------------------
1. Open: https://vercel.com/signup
2. Click "Continue with GitHub"
3. After login, go to: https://vercel.com/new
4. Find your "shakespeare-genetic-algorithm" repository
5. Click "Import"
6. Click "Deploy"
7. Wait 2-3 minutes
8. DONE! Copy your live URL

Your app is now live at: https://shakespeare-genetic-algorithm.vercel.app

================================================================================
                    METHOD 3: GITHUB PAGES
================================================================================

STEP 1: Install gh-pages
-------------------------
cd shakespeare-ga

npm install --save-dev gh-pages


STEP 2: Edit package.json
--------------------------
Open package.json in a text editor

ADD this line after "name" (near the top):
  "homepage": "https://YOUR-USERNAME.github.io/shakespeare-genetic-algorithm",

ADD these lines in the "scripts" section:
  "predeploy": "npm run build",
  "deploy": "gh-pages -d build"

Save the file


STEP 3: Push to GitHub
-----------------------
git init
git add .
git commit -m "Initial commit"

Create repo on GitHub: https://github.com/new
- Name: shakespeare-genetic-algorithm

git remote add origin https://github.com/YOUR-USERNAME/shakespeare-genetic-algorithm.git
git branch -M main
git push -u origin main


STEP 4: Deploy
--------------
npm run deploy

Wait 2-3 minutes

Your app is live at: https://YOUR-USERNAME.github.io/shakespeare-genetic-algorithm

================================================================================
                    TROUBLESHOOTING
================================================================================

Problem: Git command fails
Solution: Make sure you're in the shakespeare-ga folder
          Run: pwd (should show path ending in shakespeare-ga)

Problem: GitHub push fails
Solution: Make sure you replaced YOUR-USERNAME with actual username
          Check you created the repository on GitHub first

Problem: Build fails
Solution: Run locally first: npm run build
          Fix any errors shown

Problem: Blank screen after deployment
Solution: Check browser console (F12) for errors
          Verify build succeeded in deployment logs

Problem: 404 error
Solution: Check vercel.json and render.yaml files exist
          For GitHub Pages, enable Pages in repo settings
